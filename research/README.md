# research/ — Background Research Notes

Source research material behind the toolkit's design decisions: microservice
architecture patterns, ScalarDB use cases and editions, data modeling, database
investigation, and related topics. Files are numbered by topic and maintained as
`_en.md` / `_ja.md` pairs.

These documents are **reference material only** — they are not read by any skill
or orchestrator at runtime. For the operative guidance skills actually consume,
see `rules/` and `skills/common/references/`.
